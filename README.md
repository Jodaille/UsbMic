# UsbMic
USB microphone with ADMP401 MEMS

## Wiring schema
![Wiring schema]( https://raw.githubusercontent.com/Jodaille/UsbMic/master/Wiring_draw.jpg "Wiring schema" )

## Hardware used
- ADMP401
- HX1117 AH 3.3HS1430
- USB Soundcard (1b3f:2008 Generalplus Technology Inc.)
